# 미팅 페이지 디자인 마이그레이션 가이드

## 개요

사용자의 모든 정산 모임을 관리하는 메인 페이지입니다. 모임정산과 간편정산을 필터링하여 볼 수 있으며, 새로운 정산을 생성할 수 있습니다.

## 레이아웃 구조

### 전체 컨테이너

- **배경색**: `#ffffff` (흰색)
- **패딩**: 좌우 `20px`
- **높이**: `100vh` (전체 화면 높이)
- **레이아웃**: Flexbox (세로 방향)

### 헤더 영역 (Sticky)

- **위치**: 상단 고정 (`position: sticky, top: 0`)
- **배경**: `#ffffff`
- **패딩**: 상단 `20px`, 하단 `0`
- **z-index**: `10`
- **구성 요소**:
    1. 상단 바 (HeaderTop)
    2. 필터 탭 (FilterTabs)

#### 상단 바 (HeaderTop)

- **레이아웃**: Flexbox (가로, 양쪽 정렬)
- **구성 요소**:
    - 사용자 이름: `{사용자명}님의 모임 👋`
        - 폰트: `20px`, `font-weight: 700`
        - 색상: `#191f28`
        - letter-spacing: `-0.3px`
    - 설정 버튼
        - 크기: `48px × 48px`
        - 배경: `#ffffff`
        - border-radius: `16px`
        - 그림자: `0 2px 8px rgba(0, 0, 0, 0.04)`
        - 호버 시: 그림자 증가, `translateY(-1px)`

#### 필터 탭 (FilterTabs)

- **레이아웃**: Flexbox (가로, `gap: 8px`)
- **스크롤**: 가로 스크롤 가능 (스크롤바 숨김)
- **패딩**: 하단 `15px`
- **탭 종류**:
    1. 전체 (active 상태에 따라 카운트 표시)
    2. 모임정산 (regular meetings 카운트)
    3. 간편정산 (simple meetings 카운트)

##### 필터 탭 스타일

- **기본 상태**:
    - 배경: `#f1f3f5`
    - 텍스트: `#6b7684`
    - 패딩: `10px 16px`
    - border-radius: `12px`
    - 폰트: `14px`, `font-weight: 600`
- **활성 상태**:
    - 배경: `#3182f6` (파란색)
    - 텍스트: `#ffffff`
    - 호버: `#2870ea`
- **카운트 배지**:
    - 배경: 활성 시 `rgba(255,255,255,0.2)`, 비활성 시 `#dee2e6`
    - 텍스트: 활성 시 `#ffffff`, 비활성 시 `#495057`
    - 패딩: `2px 8px`
    - border-radius: `10px`
    - 폰트: `12px`, `font-weight: 700`

### 미팅 리스트 영역

- **레이아웃**: Flexbox (세로, `gap: 8px`)
- **스크롤**: 세로 스크롤 가능 (스크롤바 숨김)
- **마진**: 상단 `10px`
- **모바일**: 하단 마진 `80px` (FAB 공간 확보)

## 미팅 카드 디자인

### 카드 컨테이너

- **배경**: `#ffffff`
- **패딩**: `20px 24px`
- **border-radius**: `16px`
- **테두리**: `1px solid #e6e6e666` (반투명 회색)
- **레이아웃**: Flexbox (가로, 양쪽 정렬)
- **호버 효과**:
    - 상단 그라데이션 바 표시: `linear-gradient(90deg, #3182f6 0%, #00d2ff 100%)`
    - 그림자: `0 4px 16px rgba(0, 0, 0, 0.06)`
    - `translateY(-1px)`

### 카드 내부 구성

#### 미팅 정보 (MeetingInfo)

- **레이아웃**: Flexbox (세로, `gap: 8px`)
- **정렬**: 왼쪽 정렬
- **flex**: `1` (남은 공간 차지)

##### 미팅 헤더 (MeetingHeader)

- **레이아웃**: Flexbox (가로, `gap: 8px`)
- **구성 요소**:
    1. 날짜 (MeetingDate)
        - 폰트: `13px`, `font-weight: 500`
        - 색상: `#8b95a1`
        - letter-spacing: `-0.2px`
    2. 태그 (MeetingTag)
        - 모임정산: 배경 `#dbeafe`, 텍스트 `#1d4ed8`
        - 간편정산: 배경 `#fef3c7`, 텍스트 `#d97706`
        - 폰트: `11px`, `font-weight: 600`
        - 패딩: `4px 8px`
        - border-radius: `8px`
        - letter-spacing: `-0.1px`

##### 미팅 이름 (MeetingName)

- **폰트**: `14px`, `font-weight: 600`
- **색상**: `#191f28`
- **letter-spacing**: `-0.3px`
- **최대 길이**: 15자 (초과 시 `...` 처리)

#### 액션 버튼 (ActionButtons)

- **레이아웃**: Flexbox (가로, `gap: 8px`)
- **모바일 (≤350px)**: 세로 방향, `gap: 6px`

##### 아이콘 버튼 (IconButton)

- **크기**: `44px × 44px`
- **배경**: `#f8f9fa`
- **border-radius**: `12px`
- **색상**: `#6c757d`
- **호버 효과**:
    - 배경: `#e9ecef`
    - 그림자: `0 2px 8px rgba(0, 0, 0, 0.08)`
    - `translateY(-1px)`
- **첫 번째 버튼 (수정)**: 호버 시 파란색 (`#e3f2fd`, `#1976d2`)
- **두 번째 버튼 (삭제)**: 호버 시 빨간색 (`#ffebee`, `#d32f2f`)

## 빈 상태 (EmptyState)

- **배경**: `#ffffff`
- **패딩**: `48px 24px`
- **border-radius**: `20px`
- **그림자**: `0 4px 16px rgba(0, 0, 0, 0.04)`
- **텍스트 정렬**: 중앙
- **텍스트 스타일**:
    - 첫 번째 줄: `18px`, `font-weight: 600`, `#495057`
    - 두 번째 줄: `16px`, `font-weight: 500`, `#6c757d`
    - letter-spacing: `-0.2px`

## 스켈레톤 UI

- **애니메이션**: Shimmer 효과
    - 그라데이션: `linear-gradient(90deg, #f0f0f0 0px, #e0e0e0 40px, #f0f0f0 80px)`
    - 배경 크기: `1000px 100%`
    - 애니메이션: `1.5s infinite`
- **카드 수**: 4개
- **구성 요소**:
    - 날짜/태그 영역: `80px × 13px`, `60px × 20px`
    - 이름 영역: `150px × 14px`
    - 버튼 영역: `44px × 44px` (2개)

## 플로팅 액션 버튼 (FAB) - 모바일 전용 (<768px)

### 메인 FAB 버튼

- **위치**: 고정 (`fixed bottom-6 right-6`)
- **크기**: `64px × 64px` (w-16 h-16)
- **배경**: `linear-gradient(135deg, #3B82F6 0%, #1D4ED8 100%)`
- **border-radius**: `50%` (원형)
- **그림자**: `0 8px 24px rgba(59, 130, 246, 0.4)`
- **아이콘**: Plus 아이콘 (회전 애니메이션)
    - 열림 상태: `45deg` 회전
    - 닫힘 상태: `0deg`

### FAB 메뉴 (열림 상태)

- **위치**: 메인 버튼 위쪽 (`bottom: 76px`)
- **레이아웃**: Flexbox (세로, `gap: 12px`)
- **애니메이션**:
    - 등장: `opacity: 0 → 1`, `y: 20 → 0`
    - 각 버튼: 순차적 등장 (delay: 0.05s, 0.1s, 0.15s)

#### FAB 메뉴 버튼

1. **간편 정산 만들기**

    - 배경: `linear-gradient(135deg, #F59E0B 0%, #D97706 100%)`
    - 그림자: `0 4px 16px rgba(245, 158, 11, 0.4)`
    - 아이콘: Zap (번개)
    - 텍스트: `15px`, `font-weight: 600`

2. **새로운 모임 만들기**

    - 배경: `linear-gradient(135deg, #3B82F6 0%, #1D4ED8 100%)`
    - 그림자: `0 4px 16px rgba(59, 130, 246, 0.4)`
    - 아이콘: Users (사람들)
    - 텍스트: `15px`, `font-weight: 600`

3. **여행 정산 만들기**
    - 배경: `linear-gradient(135deg, #8B5CF6 0%, #7C3AED 100%)`
    - 그림자: `0 4px 16px rgba(139, 92, 246, 0.4)`
    - 아이콘: Plane (비행기)
    - 텍스트: `15px`, `font-weight: 600`

**공통 스타일**:

- 패딩: `px-5 py-3.5`
- border-radius: `2xl` (16px)
- 호버: 그림자 증가, `scale: 0.95` (active)
- 비활성화: `opacity: 0.7`, `cursor: not-allowed`
- 아이콘 컨테이너: `w-10 h-10`, `bg-white/20`, `rounded-xl`

## 애니메이션

### 페이지 진입

- **헤더**: `opacity: 0 → 1`, `y: -20 → 0` (0.5s)
- **리스트**: `opacity: 0 → 1` (0.5s, delay: 0.2s)
- **버튼**: `opacity: 0 → 1`, `y: 20 → 0` (0.5s, delay: 0.4s)

### 카드 애니메이션

- **등장**: `opacity: 0 → 1`, `scale: 0.8 → 1` (0.2s)
- **필터 변경**: `opacity: 0 → 1`, `y: 20 → 0` (0.3s)

### 인터랙션

- **버튼 호버**: `translateY(-2px)` 또는 `scale: 0.98`
- **버튼 클릭**: `scale: 0.9` 또는 `scale: 0.98`
- **FAB 토글**: Plus 아이콘 회전 (`0deg → 45deg`)

## 주요 기능

1. **필터링**: 전체/모임정산/간편정산 필터
2. **모임 생성**: 3가지 타입 (모임정산, 간편정산, 여행정산)
3. **모임 관리**: 수정, 삭제
4. **반응형**: 모바일/데스크톱 최적화
5. **애니메이션**: 부드러운 전환 효과
